# Wedding Invitation Website

This is a simple, interactive wedding invitation website designed to showcase the event details, admin dashboard, and provide an RSVP form.

The project is live and accessible via the link below:

[Live Demo](https://wedding-web-reza-shila-2026.netlify.app/)

## Features

- **Event Details**: Displays event information like date, time, and location.
- **RSVP Form**: Allows guests to RSVP for the wedding via a form connected to Supabase.
- **Responsive Design**: Optimised for both mobile and desktop devices.
- **Dashboard Admin**: Display guest information like name and phone number.

## Technology Used

- HTML5
- CSS3
- JavaScript
- Supabase
- Netlify (for hosting)

## How To Use

1. Fork this repository.
2. Edit HTML, CSS, and JavaScript files
3. Set up Supabase
4. Push your changes and deploy to Netlify.
